Contextual: Peer39

PLEASE NOTE: If you are adding a channel that is not currently in the inclusion/exclusion bucket, the tool will add the channel without overriding any previous targeting. However, if you are adding a channel to the inclusion bucket that is currently in the exclusion bucket, the tool will automatically remove the channel from the inclusion bucket and then add it to the exclusion bucket (and vise-versa for adding to the exclusion bucket) – you do not need to run the tool with 'remove from' first, and then re-run the tool with the new targeting.
  Peer39 Inclusion Logic:
**The 'CURRENT' logic setting applies the logic that is currently being used in the Peer39 inclusion bin

**Exclusion logic is always 'OR'
